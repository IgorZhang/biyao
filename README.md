### 框架说明

总共有四个大分类：每个分类中都有相应的html.css.js等文件夹，最外层有公共的JS文件，请将调用路径写正确。

本次项目主要是是采用了angularJS进行架构。使用了JQ,swiper,等等几个三方库混合使用。使用angularJS的$location服务进行url的取参，用H5最新的localStroage进行数据的缓存，模拟数据的的作用。
